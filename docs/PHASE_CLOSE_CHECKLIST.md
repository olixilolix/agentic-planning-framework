# Phase Close Checklist

## Purpose

This checklist is the manual phase-close protocol for a repo using the Agentic Planning Framework Kit.

It exists to enforce discipline without auto-commits or auto-pushes.

## Required Steps

1. Update the truth docs.
2. Record decisions.
3. Record unresolveds.
4. Verify alignment between code, docs, and agent rules.
5. Run the smallest useful verification set.
6. Review repo state.
7. Write a short handoff summary.
8. Make one clean commit.
9. Push intentionally, not automatically.

## Minimum Deliverables Before Saying "Phase Complete"

Before claiming a phase is complete, there should be:

- updated docs
- explicit unresolveds
- verification notes
- clear next step

## Working Rule

There is no assumption of automatic doc sync or automatic push.
