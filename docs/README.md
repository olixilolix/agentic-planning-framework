# Framework Docs

This folder contains the reusable workflow and governance spine of the Agentic Planning Framework Kit.

These docs are intentionally generic. They should tell a repo how to operate work without becoming a substitute for that repo's product architecture.

## Suggested Reading Order

1. [DEVELOPMENT_FRAMEWORK.md](DEVELOPMENT_FRAMEWORK.md)
2. [DECISION_GATES.md](DECISION_GATES.md)
3. [TEST_CHARTER.md](TEST_CHARTER.md)

## File Guide

- [DEVELOPMENT_FRAMEWORK.md](DEVELOPMENT_FRAMEWORK.md): canonical workflow model, pre-phase questions, milestone emphasis, closeout rules, donor adoption, improvement proposals, and `AGENTS.md` alignment
- [DECISION_GATES.md](DECISION_GATES.md): what an agent may assume safely versus what must be escalated
- [TEST_CHARTER.md](TEST_CHARTER.md): quality bar by layer

## Working Rule

These docs define how work is framed, gated, verified, and closed.

They do not replace the project-specific architecture, roadmap, or backlog.
